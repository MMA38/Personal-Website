Calculator with user interface

Sep 2024

OOP (Java), Ottawa

Designed and developed a simple calculator application using Java with a Graphical User Interface (GUI) built with Swing. The calculator performs basic arithmetic operations, such as addition, subtraction, multiplication, and division. The GUI features interactive buttons for digits and operations, along with a clear display for input and results. The project followed Object-Oriented Programming (OOP) principles, ensuring modular code structure with separate classes handling the user interface, logic, and event handling. The GUI was made responsive and user-friendly, providing real-time updates as users interact with the calculator.
